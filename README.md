## 🔥 🚀 Featured Projects

### 🩸 Hemo AI — Intelligent Blood Report Analyst
An advanced AI-powered web application designed to parse, analyze, and interpret complex CBC (Complete Blood Count) and metabolic panels into easy-to-understand, patient-friendly insights.

*   **Smart Document Parsing:** Uses OCR and Language Models to instantly extract biomarkers and reference ranges from uploaded PDF or image lab reports.
*   **Predictive Health Insights:** Flags abnormal levels (e.g., Low Hemoglobin, High WBC) and provides contextual explanations of potential underlying causes.
*   **Secure Backend:** Engineered with high-performance endpoints to ensure user medical uploads remain confidential and securely processed.
*   **Tech Stack:** `Python`, `FastAPI`, `PostgreSQL`, `SQLAlchemy`, `Google Gemini API`, `React.js`
